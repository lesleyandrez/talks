View Ports

e os pixels

Para o web design existem basicamente dois tipos de pixels muito úteis

  • CSS pixel
  • pixel físico
E para trabalhar com o CSS pixel precisamos usar a viewport
<meta name="viewport" content="width=device-width,initial-scale=1,user-scalable=no">
